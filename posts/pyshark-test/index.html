Introduction

This is a first look at pyshark. Since I'm mostly interested in WiFi I'm going to look at that here. Before this code is run I put it in monitor mode using airmon-ng a shell-script to help you do this. The interface says that tshark should be able to put it into monitor mode but it didn't work when I tried it, but this is the first time I've tried it so I might have missed something.
